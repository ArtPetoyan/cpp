## Getting Started

To get started with this repository:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ArtPetoyan/cpp.git
   cd cpp
   ```

2. **Build and Run:**
   Most examples can be compiled using `make`:
   ```bash
   cd cpp0X/ex0X
   make
   ./example
   ```
   Replace `cpp0X/ex0X` with the name of the file you want to build.
